# Installation Instructions

## Prerequisites
1. Ensure you have a CC: Tweaked computer running.
2. Make sure you have access to the in-game terminal.

## Step-by-Step Instructions

### Step 1: Access Your Computer
- Right-click on your CC: Tweaked computer.
- Select 'Open' to access the terminal.

### Step 2: Create a New File for the Loader
- In the terminal, type the following command to create a new file named `loader.lua`:
  ```
  edit loader.lua
  ```
- This opens the text editor for the new file.

### Step 3: Copy the Loader Code
- Visit the source where the `loader.lua` code is hosted.
- Select all the code in the `loader.lua` file and copy it to your clipboard.

### Step 4: Paste the Code into Your Computer
- Go back to the terminal where you have the `loader.lua` file open.
- Right-click and select 'Paste' (or use the `Ctrl + V` command) to paste the copied code into the editor.

### Step 5: Save Your File
- To save the file in the editor, press `Ctrl + S`.
- You can then exit the editor by pressing `Ctrl + W`.

### Step 6: Run the Loader
- Back in the terminal, you can run the loader by typing:
  ```
  lua loader.lua
  ```
